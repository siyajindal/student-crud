## Installation

1. Clone repository

```shell
 https://github.com/siyajindal/student-crud.git
```

## Install Client (React Files)

2. Get in the client folder

```shell
cd client
```

3. Install dependencies via npm or yarn

```shell
npm i
```

4. Start Client

```shell
npm start
```

## Install Server (Node Files)

2. Get in the server folder

```shell
cd server
```

3. Install dependencies via npm or yarn

```shell
npm i
```

4. Start Server

```shell
nodemon server
```
